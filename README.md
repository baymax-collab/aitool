# 介绍
根据剧本，可以自动切割分镜，分镜可以由 GPT 自动转换为图像提示词，无需手动写提示词。根据提示词，批量生成图片。

AI绘图工具选择：
- 可以使用本地的Stable Diffusion
- 也可以使用GPT的DLL3模型
